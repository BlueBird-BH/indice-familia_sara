# Familia Sara
![Diagrama](Diagrama.png?raw=true "Title")
La familia Sara es un grupo de bots con el proposito de actuar como asistentes personales. Cada proyecto tiene un rol de "madre" o "hija". Las madres actuan como las bases, y las hijas nacen apartir del codigo de una madre. Cada proyecto esta escrito principalmente en Python 3

## Instalación
**Descargue la última versión**
```
git clone https://github.com/BlueBird-BH/familia-sara.git
```
**Cambie al directorio de la herramienta**
```
cd familia-sara
```
**Inicie ‘Blue Password Generator’**
```
python sara.py

[-] O si usted tiene instalada más de una versión de Python...

python3 sara.py
```

## Requisitos
* Python 3 (Se recomienda la versión 3.9).
